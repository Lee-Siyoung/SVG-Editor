# SVG Editor

A lightweight, browser-based SVG editor — no installation, no sign-up required.

**[Live Demo →](https://createtosvg.netlify.app)**

---

## Features

- **Drawing tools** — Path, Rectangle, Circle, Ellipse, Line, Polygon
- **Import SVG** — Paste or upload any SVG file and edit it immediately
- **Export** — Save as SVG, PNG, or JPG
- **Properties panel** — Adjust fill, stroke, opacity, corner radius, and transform
- **SVG code editor** — View and edit raw SVG code directly
- **Undo / Redo** — Full history support
- **Grid & Zoom** — Toggle grid, zoom in/out, pan with hand tool
- **Multilingual** — Korean, English, Japanese, Chinese

## Keyboard Shortcuts

| Key | Tool |
|-----|------|
| `V` | Select |
| `H` | Hand (pan) |
| `P` | Path |
| `R` | Rectangle |
| `O` | Circle |
| `E` | Ellipse |
| `L` | Line |
| `G` | Polygon |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Space + drag` | Pan |
| `Mouse wheel` | Zoom |
